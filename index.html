Use Manual

Introduction:

Visualization Part:

This part shows cells. Drag to move camera. Click to select cell.


Children Tree Part:

This part shows cells and there's child. Drag to move. Click to select. Click with ctrl button can jump to the time when the cell first appears.


Control Part:

This part is function part. The first slide can change time. The first line buttons are playing controls. The second line buttons are Color controls. They can color the cell by celltype or gene expression. The third line buttons can show Neighbor or offsprings of selected cells. The last line can select cells by names(Space to split) or neighbor's information. Below this is a color palette which can color selected cells.


Info Part:

This part shows information of selected cells.


Gene Change Part:

This part can change tracking genes. Click "Change Genes" button on the right of top and then select gene from opened table.
